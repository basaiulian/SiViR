1. Introduction

SiViR(Similar Video Retriever) is a Web Application that offers you the possibility of searching videos on multiple platforms such as: YouTube, Vimeo, Instagram.

2. Technologies used

2.1 HTML5

HTML 5 is a markup language used for structuring and presenting content on the World Wide Web. It was the fifth and last major version of HTML that is a World Wide Web Consortium (W3C) recommendation.

2.2 CSS

CSS stands for "Cascading Style Sheet." Cascading style sheets are used to format the layout of Web pages. They can be used to define text styles, table sizes, and other aspects of Web pages that previously could only be defined in a page's HTML.

2.3 PHP

PHP (recursive acronym for PHP: Hypertext Preprocessor) is a widely-used open source general-purpose scripting language that is especially suited for web development and can be embedded into HTML.

2.4 XML

XML stands for "Extensible Markup Language.". XML is used to define documents with a standard format that can be read by any XML-compatible application.

2.5 JavaScript

JavaScript(JS) is a programming language commonly used in web development. It is a lightweight, interpreted, or just-in-time compiled programming language with first-class functions.

2.6 AJAX

AJAX (Asynchronous JavaScript and XML) allows web pages to be updated asynchronously by exchanging data with a web server behind the scenes. This means that it is possible to update parts of a web page, without reloading the whole page.

2.7 MySQL

MySQL is a freely available open source Relational Database Management System (RDBMS) that uses Structured Query Language (SQL). SQL is the most popular language for adding, accessing and managing content in a database.

2.8 HTTP Methods

There are two ways the browser client can send information to the web server:

ALINIAT -> The GET method sends the encoded user information appended to the page request.

ALINIAT -> The POST method transfers information via HTTP headers.

2.9 REST

Representational state transfer (REST) is a software architectural style that defines a set of constraints to be used for creating Web services. Web services that conform to the REST architectural style, called RESTful Web services, provide interoperability between computer systems on the Internet.

3. Application architecture

The Model-View-Controller (MVC) is an architectural pattern that separates an application into three main logical components: the model, the view, and the controller. Each of these components are built to handle specific development aspects of an application.

3.1 Model

The central component of the pattern. It is the application's dynamic data structure, independent of the user interface. It directly manages the data, logic and rules of the application.

3.2 View

Any representation of information such as a chart, diagram or table. Multiple views of the same information are possible, such as a bar chart for management and a tabular view for accountants.

3.3 Controller

The controller is that part of the application that handles the user interaction. The controller interprets the mouse and keyboard inputs from the user, informing model and the view to change as appropriate.

4. Implementation

The index_style.php has the role of formatting HTML from the following PHP documents: top.php, top_search.php, home.php, bottom.php.

The search_style.php has the role of formatting HTML from search.php document.

In controller.php the user input is taken being used in requests to YouTube API, Vimeo API, Instagram API(via php-instagram-scrapper). The results obtained from each API are stored in an array to be retrieved, verified, and displayed in search.php.

The config.php contains the information essential for the proper functioning of the database and the APIs.

In search.php the information from the array with results following requests to the API is taken over, inserted in the database and displayed, giving the user the ability to view media content directly on the page or via an RSS feed.

In model.php the connection to the database is established. Also, queries, inserts and data deletions are performed.
